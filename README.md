# 使用方式
复制物品ID后到自定义代码表格的道具ID列里右键选择“粘贴为数值”<br>
然后选择E列直接复制粘贴到金手指文件上，建议用<a href="https://www.editplus.com/">EditPlus</a>工具改为UTF-8编码

# NHSE是什么？
存档修改工具，个人单纯做用修改名字以及修改住民用，目前并没具体数据对应修改住民<br>
打开JKSV导出的存档文件夹读取即可<br>
